# 货架管理模块部署说明

## 📦 已完成的工作

### 1. 后端控制器

**文件路径**：`d:\xampp\htdocs\crmeb\app\adminapi\controller\v1\shelf\Shelf.php`

**功能说明**：

- ✅ `index()` - 货架列表页面（支持搜索、筛选、分页）
- ✅ `stock()` - 库存管理页面（库存查询、预警设置、补货操作）
- ✅ `report()` - 销售报表页面（销售统计、商品排行、数据导出）

**继承关系**：

- 继承自 `\app\adminapi\controller\AuthController`
- 自动继承权限验证、管理员信息、请求处理等功能

**数据操作**：

- 使用 `Db::name('eb_xxx')` 进行数据库操作
- 使用 `$this->request->param()` 获取请求参数

## 🚀 部署步骤

### 第一步：安装数据库表

**方法一：使用 phpMyAdmin（推荐）**

1. 访问 `http://localhost/phpmyadmin`
2. 选择 `crmeb` 数据库
3. 点击"导入"选项卡
4. 点击"选择文件"，选择：`d:\xampp\htdocs\crmeb\install_shelf_database.sql`
5. 点击"执行"
6. 等待导入完成

**方法二：使用命令行**

```bash
cd d:\xampp\mysql\bin
mysql -u root -p crmeb < d:\xampp\htdocs\crmeb\install_shelf_database.sql
```

## 🔧 关联的数据表

### 主表

- `eb_physical_shelf` - 物理货架表
- `eb_shelf_stock` - 货架商品库存表
- `eb_shelf_replenish` - 货架补货记录表

### 关联表

- `eb_area` - 区域管理表
- `eb_distributor` - 配送员表
- `eb_store_order` - 订单表（扩展了货架相关字段）
- `eb_user` - 用户表（扩展了区域、欠费等字段）
- `eb_water_ticket` - 水票表
- `eb_water_ticket_log` - 水票核销记录表
- `eb_system_remind` - 系统提醒表

## 🔐 权限配置说明

### 权限标识

| 权限标识              | 说明         | 对应方法 |
| --------------------- | ------------ | -------- |
| admin-shelf-index     | 查看货架列表 | index()  |
| admin-shelf-stock     | 库存管理     | stock()  |
| admin-shelf-report    | 销售报表     | report() |
| admin-shelf-add       | 添加货架     | -        |
| admin-shelf-edit      | 编辑货架     | -        |
| admin-shelf-del       | 删除货架     | -        |
| admin-shelf-replenish | 补货操作     | -        |

### 配置方法

1. 登录后台
2. 进入【系统管理】→【权限管理】
3. 选择需要授权的角色
4. 在权限列表中找到"货架"菜单
5. 勾选对应的权限
6. 保存配置
