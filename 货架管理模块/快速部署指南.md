# 🚀 货架管理模块快速部署指南

## ⚡ 3 分钟快速部署

### 方法一：一键自动安装（推荐）

**步骤**：

1. **访问安装页面**

   ```
   http://localhost/crmeb/install_shelf_module.php
   ```

2. **点击"立即安装"按钮**

   - 系统会自动创建数据库表
   - 自动配置菜单
   - 自动插入测试数据

3. **等待安装完成**

   - 进度条显示安装进度
   - 安装成功后自动跳转到后台

4. **删除安装脚本**（重要！）
   ```
   删除文件：d:\xampp\htdocs\crmeb\install_shelf_module.php
   ```

---

### 方法二：手动安装

#### 步骤 1：安装数据库表

**使用 phpMyAdmin**：

1. 访问 `http://localhost/phpmyadmin`
2. 选择 `crmeb` 数据库
3. 点击"导入"
4. 选择文件：`install_shelf_database.sql`
5. 点击"执行"

**或使用命令行**：

```bash
cd d:\xampp\mysql\bin
mysql -u root -p crmeb < d:\xampp\htdocs\crmeb\install_shelf_database.sql
```

#### 步骤 2：配置菜单

1. 打开 phpMyAdmin
2. 选择 `crmeb` 数据库
3. 点击"SQL"选项卡
4. 打开文件：`install_shelf_menus.sql`
5. 复制内容并粘贴到 SQL 输入框
6. 点击"执行"

#### 步骤 3：验证安装

```sql
-- 检查表是否创建成功
SHOW TABLES LIKE 'eb_%';

-- 检查菜单是否配置成功
SELECT * FROM eb_system_menus WHERE menu_name IN ('货架', '货架列表', '库存管理', '销售报表');
```

## 🔧 常见问题

### 问题 1：菜单不显示

**原因**：菜单未配置或权限不足

**解决方法**：

1. 检查菜单是否已添加到数据库
   ```sql
   SELECT * FROM eb_system_menus WHERE menu_name IN ('货架', '货架列表', '库存管理', '销售报表');
   ```
2. 检查管理员权限配置
3. 清除缓存：删除 `runtime/cache/*`

### 问题 2：页面显示空白

**原因**：模板文件路径错误或语法错误

**解决方法**：

1. 检查视图文件是否存在
   ```
   app/adminapi/view/shelf/index.html
   app/adminapi/view/shelf/stock.html
   app/adminapi/view/shelf/report.html
   ```
2. 开启调试模式查看错误信息
3. 检查浏览器控制台错误

### 问题 3：数据库连接失败

**原因**：数据库配置错误

**解决方法**：

1. 检查 `.env` 文件中的数据库配置
2. 确认数据库名、用户名、密码正确
3. 确认数据库服务已启动

### 问题 4：权限不足

**原因**：未分配权限

**解决方法**：

1. 登录后台
2. 进入【系统管理】→【权限管理】
3. 选择角色
4. 勾选"货架"相关权限
5. 保存配置
